## AND (&)
```
5 & 6
5 = 0101
6 = 0110
---------
4 = 0100
```

## OR (|)
```
5 | 6
5 = 0101
6 = 0110
---------
7 = 0111
```

## XOR (^)
```
5 ^ 6
5 = 0101
6 = 0110
---------
3 = 0011
```

## NOT (~)
```
~6
6  = 0110
~6 = 1001
```

## SHIFT LEFT (<<) 
```
M << N = M * 2^N
```

## SHIFT RIGHT (>>)
```
M >> N = M / 2^N (unsigned values)
```
